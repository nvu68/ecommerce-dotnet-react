# 💻 Hi-End E-Commerce Platform

Một hệ thống Website thương mại điện tử chuyên cung cấp Laptop & PC High-end, được xây dựng theo kiến trúc Fullstack hiện đại.

## 🚀 Công nghệ sử dụng (Tech Stack)
* **Backend:** C# ASP.NET Core Web API, Entity Framework Core, SQL Server.
* **Frontend:** ReactJS, TypeScript, Tailwind CSS, Axios.
* **Authentication:** JSON Web Token (JWT) + BCrypt.

## ✨ Tính năng nổi bật (Features)
* **Client:** Xem danh mục 3 cấp, Lọc sản phẩm, Giỏ hàng (Offcanvas), Thả tim yêu thích (Context API), Checkout.
* **Admin:** Quản lý sản phẩm, Theo dõi và cập nhật trạng thái đơn hàng (Pending -> Shipping -> Completed).
* **Security:** Mã hóa mật khẩu, phân quyền truy cập Role-based (User/Admin).
